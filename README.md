# X1 Mídias — Landing Page

Landing page institucional + de captação da X1 Mídias (Alegre - ES).
Gestão de tráfego pago, redes sociais, identidade visual e sites.

## Estrutura dos arquivos

```
x1midias-deploy/
├── index.html          # Página principal (com SEO completo no <head>)
├── logo.png            # Logo da X1 Mídias
├── og-image.png        # Imagem 1200x630 para preview em redes sociais
├── favicon.png         # (alias do logo.png)
├── sitemap.xml         # Mapa do site para Google
├── robots.txt          # Instruções para bots de busca
└── cases/
    ├── case1.mp4       # Vídeo case Saúde
    ├── case2.mp4       # Vídeo case Imóveis
    ├── case3.mp4       # Vídeo case Locação
    └── case4.mp4       # Vídeo case Serviços
```

## Personalização pós-deploy

Depois de subir, **edite o `index.html`** e troque:
- `https://lp.anunciex1midias.com.br/` pelo seu domínio real (em 7 lugares: canonical, og:url, og:image, twitter:image, Schema.org URL/logo/image)
- O `sitemap.xml` e `robots.txt` (URL do Sitemap)
